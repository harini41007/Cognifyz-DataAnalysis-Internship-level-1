# Cognifyz-DataAnalysis-Internship-level-1
This repository contains Level 1 tasks completed during my Cognifyz Technologies Data Analysis Internship, including cuisine analysis, city-wise restaurant analysis, price range distribution, and online delivery analysis using Python."
 ## Objective
 The objective of this project is to analyze restaurant data and extract meaningful insights using python. The analysis focuses on cuisines,cities,price ranges, and online delivery services offered by restaurants.

 ## Dataset Overview
 The dtaset contains information about restaurants,including:
 + Restaurant Name
 + City
 + Aggregate Rating
 + Price Range
 + Online Delivery Availabiity
 + Other restaurant-related details

# Tools & Technologies
+ Python
+ Pandas
+ NumPy
+ Matplotlib
+ Seaborn
+ Google Colab
+ Github

## Tasks Performed

### Task 1: Top Cuisines Analysis
#### Steps 
1. Loaded the dataset
2. Cleaned missing values in the cuisines column
3. Identified the top three most commom cuisines
4. Calculated cuisines percentages
5. visualized the results using bar and pie chart

### Task 2: City Analysis
#### Steps
1. Counted restaurants in each city
2. Identified the city with the highest number of restaurants
3. Calculated average ratings for each city
4. Determined the city with the highest average rating.

### Task 3: Price Range Distribution
#### Steps 
1. Analyzed restaurant price range categories
2. Calculated the percentage distribution of each category.
3. Created visualization to represent the distribution

### Task 4: Online Delivery Analysis
#### Steps
1. Calculated the percentage of restaurants offering online delivery
2. Compared average ratings of restaurants with and without online delivery
3. Visualized the comparison using a bar chart

## Outcome
+ identified the most popular cuisines among restaurants.
+ Determined cities with the highest restaurant presence and ratings
+ Analyzed the impact of online delivery services on restaurant ratings
+ Improved skills in Data Cleaning,Data Analysis, and Data Visualization using python
